Privacy Policy

Personal Data

Some browser extensions at gchristensen.github.io may ask users to provide the following personal information to perform their functions:

  • Add Torrent To:
    • Authorization credentials of qBittorrent Web UI.
    • Authorization credentials of uTorrent Web UI.
  • iShell:
    • API key used by Bing Microsoft Translator
    • API key used by Lingvo Live translation service.
    • API key used by YouTube search.
    • API keys used by Google search.

All the data listed above is stored in extension local storage memory and is not transferred to any third-party services.

Applications at gchristensen.github.io do not collect, share, or sell any other user's personal data or metadata.
